# Claude Code Glossary

A comprehensive list of terms, commands, and concepts used in Claude Code.

---

## A

**Agent** - An AI instance that can perform tasks autonomously within Claude Code

**Async Tasks** - Tasks that run in the background without blocking the main session

## B

**Bash Mode** - A mode where Claude Code can execute terminal commands directly

## C

**Claude Code** - Anthropic's command-line interface tool for AI-powered development

**Context** - The information and files that Claude has access to during a session

**Compact** - Reducing the size of conversation history while preserving key information

## F

**Forking** - Creating a new session branch from an existing session

## M

**MCP (Model Context Protocol)** - A protocol for integrating external tools and data sources

## P

**Planning Mode** - A mode where Claude creates detailed plans before executing tasks

## S

**Subagent** - A specialized AI agent spawned to handle specific subtasks

**Skill** - A reusable workflow or template in Claude Code

**Session** - A conversation thread with Claude Code

**Slash Command** - Commands that start with `/` to control Claude Code behavior
